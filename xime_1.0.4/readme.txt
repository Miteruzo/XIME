● 初めに

　この度は、『ХIME Is Music Environment!!』(以下 ХIME) を Download していただ
　き、誠にあり難う御座います。
　ХIME は MML と呼ばれる言語を使って、曲の演奏が出来る環境であります。
　MML のコマンドに就いては、help.exe を御覧ください。
　この Soft は Windows XP ～ Windows 10 で動作します。


● File 構成

　sample\	(MML のサンプル。)
　help.exe	(MML コマンドのヘルプ。)
　mmlplay.exe	(MML を再生するためのアプリケイション。削除しないでください。)
　readme.txt	(この説明書。)
　strxtprn.exe	(印刷プログラム。HSP-NEXT 様より拝借。)
　xime.exe	(ХIME 専用エディタ。)
　*.dll		(アプリケイションの動作に必要。絶対に消さないでください。)


● インストール方法

　xime_1.0.3.zip を解凍する事でインストール出来ます。


● アンインストール方法

　解凍したディレクトリを削除する事でアンインストール出来ます。


● 関聯付け方法

　*.xim 形式の File をエクスプロラから右クリックして『アプリケーションから開く』
　を選択し、解凍したディレクトリ内に在る xime.exe を選択してください。


● 使用方法

　① xime.exe を開きます。
　② MML を入力するか、『File』→『L/読込』で保存せられた File を開きます。
　　(MML に就いては F1 キーを押すと表示せられるヘルプを御覧ください。)
　③『XIME』→『C/コンパイル+再生』で曲が再生せられます。
